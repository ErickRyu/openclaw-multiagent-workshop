Before you come

참가 전 준비사항

워크숍 시간을 최대한 실습에 쓰기 위해, 아래 3가지를 반드시 사전에 완료해 주세요.

1
Node.js 설치
macOS / Linux: Node.js 22 이상
Windows: WSL 설치 후 WSL 내에 Node.js 설치
확인: 터미널에서 node -v 입력 시 버전 출력
2
AI API 준비
✦ 추천
ChatGPT 구독(Plus 이상) → Codex OAuth 연결
별도 API 키 발급 없이 오픈클로와 바로 연결 가능해 가장 간편합니다

추가로 Claude API 키가 있으면 더 좋습니다
에이전트별로 다른 모델을 배정하는 실습에서 활용합니다
3
오픈클로(OpenClaw) 설치
공식 문서를 참고하여 설치해 주세요:
👉 openclaw.ai
확인: openclaw --version 입력 시 버전 출력
# 환경 확인
$ node -v
v22.14.0

# 오픈클로 설치 → https://openclaw.ai 참고

# 버전 확인
$ openclaw --version
v2026.3.20
💡 오픈클로 설치가 잘 안 되셔도 괜찮습니다! 1주차 시작 시 함께 점검하고 트러블슈팅하는 시간이 있으니 부담 없이 오세요. 최소한 Node.js와 ChatGPT 구독(또는 다른 AI API)만 준비해 주시면 됩니다.

FAQ

자주 묻는 질문

개발자가 아닌데 참여할 수 있나요?
터미널 기본 조작(명령어 입력, 복사/붙여넣기)이 가능하고, 사전 준비사항을 완료할 수 있다면 참여 가능합니다. PM이나 기획자 분들도 환영합니다.
오픈클로를 써본 적이 없어도 되나요?
네, 1주차에서 기초부터 시작합니다. 다만 사전에 설치까지는 완료해 주셔야 합니다.
API 사용 비용은 얼마나 드나요?
ChatGPT 구독(Plus $20/월)이 있으면 Codex OAuth로 연결해 추가 비용 없이 실습 가능합니다. Claude API를 함께 사용하더라도 실습 중 발생하는 비용은 수천 원 이내입니다.
노트북은 꼭 가져가야 하나요?
네, 각자 노트북에 직접 세팅하는 핸즈온 워크숍입니다. macOS, Windows(WSL), Linux 모두 가능합니다.
환불 규정은 어떻게 되나요?
워크숍 3일 전까지 전액 환불 가능하며, 이후에는 환불이 어렵습니다.
